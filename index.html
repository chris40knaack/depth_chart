Special Teams

Punt Tools

Browser tools for building and sharing the shield-punt depth chart. Nothing is uploaded — your roster and lineup stay on your own device.

Offense →

Base offensive personnel — O-line, QB, backs, and receivers.

Defense →

Base 4-3 — D-line, linebackers, corners, and safeties.

Shield Punt →

Shield punt protection — line, shield, and punter.

Punt Block / Return →

Punt rush and return — rushers, jammers, and the returner.

Kickoff Coverage →

Coverage unit — L1–L5, R1–R5, and the kicker.

Kickoff Return →

Return unit — front line, mid blockers, and two returners.

PAT / Field Goal →

Kick protection — line, wings, holder, and kicker.

Roster Editor →

Edit athlete parameters — number, grade, active status, and combine metrics. Export an updated athletes.json.
First time? These tools open empty. Use Import athletes (Punt Designer) or Import JSON (Roster Editor) to load the data file you were given. It's remembered on this device for next time. To share a lineup, use Save assignments and send the file.
